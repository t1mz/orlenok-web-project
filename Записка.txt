Сайт написан с нуля. Был нарисован черновик шаблона на бумаге, затем на его основе был написан голый скелет дизайна используя только html разметку, после чего с помощью css был придан более похожий на черновик вид. 
Всего на сайте 11 страниц, включая: 8 страниц материала по теме (Введение, 6 страниц материала, Заключение), 1 страница оглавления, 1 страница для обратной связи, 1 страница для ошибки несуществующей страницы 404.
На сайте используется HTML5, т.к. в его структуре использованы семантические элементы <HEADER>, <NAV>, <MAIN>, <ARTICLE> и <FOOTER>. В связи с этим, сайт может некорректно отображаться в старых веб-браузерах.
На сайте используется JavaScript, его можно найти в виде интерактивных всплывающих изображений на главной странице сайта (index.html)
На сайте использвуется php, его можно найти на странице обратной связи (connect.html). PHP используется для отправки заполненной формы в mysql базу данных.
